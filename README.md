# 🏎️ F1 Driver Career Analytics Dashboard

An interactive Formula 1 analytics dashboard built with Python, Pandas, and Plotly.

The project analyzes historical F1 driver performance and provides visual insights into:

- Wins by season
- Championship progression
- Driver rankings
- Nationality distribution
- Career statistics
- Driver comparisons

## Features

### 📈 Wins by Season
View the number of races won by each driver across different seasons.

### 🏆 Championship Progression
Track how a driver's championship standings changed over time.

### 🌍 Nationality Distribution
Explore the geographical diversity of Formula 1 drivers.

### 🥇 Driver Rankings
Rank drivers using career statistics such as wins, podiums, and points.

### ⚔️ Driver Comparison
Compare two drivers side-by-side using interactive charts.

## Installation

Clone the repository:

```bash
git clone https://github.com/sparshpatil13412/f1-driver-analytics.git
cd f1-driver-analytics
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Running the Dashboard

```bash
python dashboard/app.py
```

Then open:

```text
http://127.0.0.1:8050
```

in your browser.
